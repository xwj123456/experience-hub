# Roadmap

Experience Hub is intentionally conservative: provenance, isolation, and explicit adoption come before autonomy.

## 0.2 — evaluation and interoperability

- Publish domain-oriented benchmark packs and blinded human-evaluation guidance.
- Define stable import/export envelopes for portable experiences and evidence snapshots.
- Add observability hooks and performance baselines for larger local datasets.
- Expand retrieval evaluation beyond committed synthetic fixtures.

## 0.3 — production integration boundary

- Document external authentication and authorization integration patterns.
- Add storage and worker extension interfaces without weakening replay guarantees.
- Provide deployment and backup examples for trusted private environments.

## Research track

- Consolidation and interference models beyond simple temperature decay.
- Outcome-linked feedback that updates confidence without rewriting history.
- Human-reviewed novelty, usefulness, falsifiability, and evidence-fit evaluation.
- Privacy-preserving multi-agent knowledge exchange.

Roadmap items are directional, not commitments. Open an issue before starting a large item so its invariants and scope can be agreed first.
